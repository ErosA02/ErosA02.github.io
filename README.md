# Co2FeSn DFT Dashboard

Interactive supplemental dashboard for the scientific poster:

**Efecto del acoplamiento espin-orbita en la estructura electronica de fases Heusler Co2FeSn**

The site contains real Quantum ESPRESSO outputs processed from the local `Poster/Co2FeSn` calculations:

- SP and SOC band structures aligned to `E - EF`
- SP/SOC density of states with focus near `EF`
- formation energies computed from elemental references
- primitive-cell structure viewer
- interactive views based on the processed plot data used by the dashboard

Author: Eros Fabian Acosta Vazquez, Universidad Autonoma de Nuevo Leon, Facultad de Ciencias Fisico Matematicas.
